设计理念

Apple Native · Subtle Tech

以 Apple Human Interface Guidelines 为基底，保持 iOS 原生的干净与克制。 在关键体验节点加入微妙的科技感——磨砂玻璃、轻盈渐变、精准动效—— 让用户感受到「这是一个精心打磨的工具」，而不是「一个 AI 生成的模板」。

三个设计原则

🍎 原生优先 — 系统控件、系统字体、系统动效，不重复造轮子

✨ 克制科技 — 磨砂玻璃只用在相机浮层，渐变只用在 CTA，不泛滥

🎯 一屏一事 — 每个页面只有一个核心操作，导航层级 ≤ 2


SF Pro 系统控件 磨砂玻璃 微渐变 极简导航 Haptic

• 拍照瞬间：屏幕白闪 0.15s + Haptic light + 快门声

• 成功 toast：磨砂玻璃胶囊 "第24周 ✓ 已保存"，底部滑入 0.3s spring

🗺 导航架构

信息架构（极简两层）

无 TabBar、无侧栏。NavigationStack + sheet 组合。

欢迎 → 预产期 → 📸 相机 (Root)
                     ├── 📅 时光轴 (push)
                     │     └── ✨ 视频预览 (sheet)
                     │           └── 💬 分享 (system share)
                     ├── ⚙️ 设置 (push)
                     └── 📷 拍照 → Toast → 留在相机

核心路径：打开 → 拍照 → 看时光轴 → 生成视频 → 分享
转场：全部使用 iOS 原生 NavigationStack push + .sheet present
