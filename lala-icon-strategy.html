🎨 图标方案：去 AI 味

🍊 娜美 | 2026-04-02

问题：当前 UI emoji 太多（💩🏠📊💬👤🔥🎯🍌…），看起来像 AI 生成的。
方案：emoji 只给便便角色，其他全部换成正经图标。

核心原则

1. 是便便角色或形态？ → 保留 emoji
2. 是 UI 导航/功能图标？ → SF Symbols
3. 是品牌关键元素？ → 手绘涂鸦
4. 不确定？ → 默认 SF Symbols

手绘图标怎么做

MVP 推荐：Figma + 手写笔快速画
风格：粗线条、不完美的手绘感、棕色墨水
数量：约 5-8 个核心元素
格式：导出 SVG → SwiftUI 直接用

备选：AI 生成 + 手动调整
Prompt: "hand-drawn doodle icon, brown ink on kraft paper, simple line art"

效果：看起来像真人设计的 iOS App，不像 AI 模板 ✨
