💩 拉了吗 — MVP 功能 Spec

🍊 娜美 | 2026-03-28

定位：便秘人群的社交支持 + 排便管理工具
一句话：让排便这件事变好玩、有人陪

模块 1：排便打卡 P0

首页中央大按钮「💩 我拉了！」

打卡流程

① 点击大按钮 → 弹出打卡面板
② 选 Bristol 类型（7种可爱插图）
③ 选心情（😫😐😊🎉）
④ 可选备注（≤50字）
⑤ 点「记录」→ 动画反馈 + 角色成长
⑥ 成功页 → 统计 + 分享按钮

模块 2：便便角色养成 P0

等级名称外观解锁
Lv.1小硬球灰色皱眉便便初始
Lv.2小软团棕色微笑便便累计7次
Lv.3小香肠金色+小帽子连续7天
Lv.4便便大师金色闪亮+皇冠连续30天
Lv.5彩虹便便彩虹色+翅膀连续60天
状态机制：
✅ 正常打卡 → 角色开心动画
⚠️ 1天未卡 → 表情变焦虑
❌ 3天未卡 → 角色变灰+下雨
🎉 回归打卡 → "欢迎回来"动画

模块 3：匿名树洞 P1

帖子类型：
💩 打卡分享 — "我今天拉了！Type 4！"
😫 吐槽 — "第5天了…"
💡 小妙招 — "早上空腹喝温水有用"
🎉 成就晒 — "连续7天畅通！"

交互：点赞（💩代替❤️）+ 评论（≤100字）
安全：关键词过滤 + 举报 + 人工审核

模块 4：排行榜 P1

榜单规则
🏆 今日畅通榜今天已打卡，越早越靠前
🔥 连续打卡榜连续天数排名
👫 好友榜互加好友可见（微信运动模式）

默认匿名（便便#8423），可选实名，可退出排行

模块 5：分享卡片 P1

触发：打卡成功 / 达成成就 / 周报生成
尺寸：1080×1920px（适配朋友圈/微博/小红书）
内容：角色形象 + 数据 + App 二维码

示例文案：
"我今天拉了！💩 Type 4 完美条！连续第12天~"
"本周排便5次，击败80%的用户 📊"
"便便宠物升级 Lv.3 小香肠！🎉"

页面结构

Tab图标内容
首页🏠角色 + 打卡按钮 + 今日状态
记录📊日历视图 + 统计图表
树洞💬匿名社区 feed
我的👤个人资料 + 设置 + 成就

验收标准

  • 用户可完成排便打卡（Bristol类型+心情）
  • 打卡后角色有动画反馈
  • 角色根据打卡频率进化/退化
  • 匿名树洞可发帖、点赞、评论
  • 排行榜展示今日畅通榜和连续打卡榜
  • 分享卡片可保存相册/分享微信
  • 所有数据加密存储
  • 包含医疗免责声明
